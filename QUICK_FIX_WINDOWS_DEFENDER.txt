================================================================================
QUICK FIX: Windows Defender False Positive
================================================================================

PROBLEM: Windows Defender flags PASAT.exe (and other tasks) as a virus

CAUSE: These are old Visual Basic 6 programs (from early 2000s). They're
       legitimate research software, but Windows Defender doesn't recognize them.

SOLUTION: Add an exclusion to Windows Defender

================================================================================
EASY METHOD (Automatic - Requires Administrator):
================================================================================

1. Right-click on PowerShell and select "Run as Administrator"
2. Navigate to this directory
3. Run: .\add_windows_defender_exclusion.ps1
4. Type 'Y' when asked to confirm
5. Done!

================================================================================
MANUAL METHOD (No PowerShell):
================================================================================

1. Open Windows Security
   - Press Windows key + I
   - Click "Privacy & Security" → "Windows Security"
   - Click "Virus & threat protection"

2. Add Exclusion
   - Click "Manage settings" under "Virus & threat protection settings"
   - Scroll down to "Exclusions"
   - Click "Add or remove exclusions"
   - Click "Add an exclusion" → "Folder"

3. Select This Folder
   - Navigate to wherever you installed CAPER_suite
   - Select the entire CAPER_suite folder
   - Click "Select Folder"

4. Done!
   - The CAPER_suite folder will now be excluded from scans
   - You can run all tasks without interference
